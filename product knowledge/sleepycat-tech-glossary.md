## 1. Cooling Technologies

### CoolTec™ Fabric / CoolTEC™
SleepyCat's branded cooling fabric used in both mattress covers and pillows. Draws heat away from the sleeper.
- **Cooling claim**: up to 4°C cooler (mattresses & pillows), up to 6°C cooler (Founder's Mattress)
- **Products using**:
  - Mattresses: The Founder's Mattress, Ultima Memory Foam Mattress, Ultima Natural Latex Mattress
  - Pillows: CoolTEC Memory Foam Pillow, Contour CoolTEC Memory Foam Pillow
  - Toppers/Covers: Mattress Cover
- **Naming variants on site**: `CoolTec™ Fabric`, `CoolTEC™`

### Ventilation Channels
Engineered airflow channels that allow air to circulate freely, reducing heat and moisture build-up.
- **Products using**: Aeroflow Foam Topper

### Cooling Layer
Page references the Aeroflow Transition layer as a dedicated cooling sub-layer in some mattresses.
- **Products using**: Cloud Mattress

## 2. Memory Foam Technologies

### SmartCell™ Memory Foam
**Patented** memory foam with a "micro-cell design" for pressure relief and airflow channels. Marketed as "India's 1st and only" patented memory foam mattress technology.
- **Products using**: The Founder's Mattress *(exclusive)*
- **Key claim**: Patented, sleeps up to 6°C cooler, integrated airflow channels
- **Use in content**: Anchor tech for the Founder's Mattress — always cite as "Patented SmartCell™"

### AirGen™ Memory Foam
100% open-cell memory foam designed for enhanced breathability, airflow, and heat reduction.
- **Products using**:
  - Mattresses: Hybrid Latex Mattress, Original Mattress, Ultima Memory Foam Mattress
  - Toppers: Memory Foam Mattress Topper, 2 Inch Memory Foam Mattress Topper
- **Naming variants on site**: `AirGen™`, `AirGen Memory Foam`, `AirGen™ Memory Foam`
- **Use in content**: Most widely used memory foam in the SleepyCat lineup — the "workhorse" memory foam vs SmartCell's "premium/patented" positioning

### HyperXD™ Foam
Adaptive foam layer that cradles without sinking. Used as the top comfort layer.
- **Products using**: The Founder's Mattress *(exclusive)*

### DeepTouch™ Pressure Technology / 7-Zone DeepTouch™ Pressure Tech
Contouring support that targets specific body parts to maintain a neutral spine position.
- **Products using**: Ultima Memory Foam Mattress, Ultima Natural Latex Mattress
- **Variants**: `DeepTouch™ Pressure Technology`, `7-Zone DeepTouch™ Pressure Tech`

### SoftTouch
SleepyCat's branded perforated memory foam pillow line.
- **Products using**: SoftTouch Memory Foam Pillow
- (Also informs the line name "Contour SoftTouch Memory Foam Pillow")

### Perforated Memory Foam
Memory foam with perforation pattern that provides consistent support without sagging.
- **Products using**: Contour CoolTEC Memory Foam Pillow

## 3. Latex Technologies

### Pinhole Tech Latex / Pinhole Technology / Pinhole Design
Natural latex with engineered pinhole pattern allowing constant airflow, preventing heat and moisture build-up.
- **Products using**:
  - Mattresses: Hybrid Latex Mattress, The Latex Ortho, Ultima Natural Latex Mattress
  - Toppers: Natural Latex Mattress Topper
- **Naming variants on site**: `Pinhole Tech Latex`, `Pinhole Technology`, `Pinhole Design`

### GOLS Certified Natural Latex
Certified natural latex layer (GOLS = Global Organic Latex Standard). Counts as both a material and a trust marker.
- **Products using**:
  - Mattresses: The Founder's Mattress, Cloud Mattress, Ultima Natural Latex Mattress, Baby Natural Latex Mattress
- **Use in content**: Lead with the GOLS cert for content targeting "organic", "natural", "non-toxic" search intent.
